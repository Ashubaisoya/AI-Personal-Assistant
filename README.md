# AI Personal Assistant

An AI-powered web application built with **Flask** and **Python**, powered by the **Groq API** for question answering and email summarization.

## Features

- Ask AI any question
- Summarize emails
- Fast responses with the Groq API
- Simple and responsive UI

## Tech Stack

- Python
- Flask
- HTML
- CSS
- JavaScript
- Groq API

## Setup

1. Clone the repository.
2. Create a `.env` file:

```env
GROQ_API_KEY=your_groq_api_key
```

3. Install the required Python packages.
4. Run:

```bash
python main.py
```

5. Open `http://127.0.0.1:5000`
